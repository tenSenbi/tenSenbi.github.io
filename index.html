경험치 계산기

도움말

  • 경험치 퍼센트 적용 해보기

    골드서버 :

    골드서버 : 골드 채널 및 스피드 서버를 말하며 각각의 배율만큼 경험치를 상승.

       
    이벤트 배수 :

    이벤트 배수는 경험치 이벤트 적용시 해당 퍼센트(%)를 씁니다.
    펫 :

    경험치 상승이 가능한 펫만 표시됩니다.

         
    반지 :

    경험치 상승이 가능한 반지만 표시됩니다.

      
    펫-반지 세트 :

    세트일 경우 체크해주세요. (가능 세트 [펫:펜더, 스켈렛톤], [반지:어둠의변신, 갈색팬더, 스켈레톤])

      
    인장 :

    버프 형태 입니다. 일반(400이하 적용.), 마렙(400이상적용.)

           
    빛의축복 :

    30분간 적용. 채널이동, 접속 종료시 적용 해제.

      
    부적 :

    구입후 인벤토리에서 적용가능.

      
    사냥지역 :

    해당 지역에 사냥시에만 체크.

      
    20주년 이벤트 :

    해당 버프 적용시 체크.
    기타 :

    특수 경험치 버프시 해당 퍼센트 입력. [아르카쟁,검투사의 명예,파티경험치 주문서](값:숫자%)
  • 현재 날짜 : -
  • 입력 내용 완료 날짜 : -
    입력 내용 남은 일수 : 0 일( - %)
  • 경험치 퍼센트 적용 날짜 : -
    경험치 퍼센트 남은 일수 : 0 일( - %)
  • 필요 경험치 : -

  • 기본 경험치 배수

    현재 경험치 (초경/%) : - / -%

    현재 기입한 경험치 내용을 표시해 줍니다.


    100% 겸험치 (초경/%) : - / 100 %

    현재 기입한 내용을 100퍼센트로 환산해 줍니다.

  • 기대 경험치

    ※기대 경험치 배수 체크를 해주세요
    예상 초당 경험치(최종퍼센트) : - (-%)
  • 최종 경험치 [배수A x 배수B] : 100%
    배수A : 100%
    + 기본 : 100%
    + 펫 : 0%
    + 반지 : 0%
    + 펫반셋 : 0%
    + 인장 : 0%
    + 빛축 : 0%
    + 부적 : 0%
    + 사냥지역 : 0%
    + 20주년 : 0%
    + 기타 : 0%
    배수B : -%(배율 : -배)
    + 골드채널 or 스피드서버: 0%
    + 이벤트 경험치 : 0%

도움말

현재 레벨 : 현재 레벨을 씁니다. (숫자만 가능.)
목표 레벨 : 도달 하고 싶은 목표 레벨을 씁니다. (숫자만 가능.)
초당 경험치 : 현재 초당 획득 경험치를 씁니다. (숫자만 가능. ※ 주의:10분이상 오토시 조금더 정확한 계산이 가능 합니다.)
현재 경험치(%) : 현재 적용된 경험치 퍼센트를 씁니다. (기대 경험치를 알기위해 정확히 적으세요.)

경험치 퍼센트 적용 해보기 : 해당 항목의 i 클릭하여 자세한 설명을 볼 수 있습니다. 각 항목에 체크 하여 향상된 경험치를 알 수 있으며, 적용된 배수에 따라 얼마나 상승하는지 알아보세요.

데이터 출력

현재 날짜 : 현재날짜가 로드됩니다.
입력 내용 완료 날짜 : 현재 입력한 내용의 완료날짜가 계산됩니다.
남은 일수 : 해당 내용의 남은일수가 계산됩니다.
경험치 퍼센트 적용 날짜 : +경험치 퍼센트를 적용한 내용을 계산해 날짜를 표시해 줍니다.
필요 경험치 : 레벨업동안 필요한 경험치를 보여줍니다.
기대 경험치 : 적용한 퍼센트로 적용한 초당 경험치를 표시해 줍니다.
